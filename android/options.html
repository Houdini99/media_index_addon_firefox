Download to Index

Configure the media index server this extension uploads to.

Root URL of your media index (no trailing slash). The login page is expected at /login and uploads at /upload.

How to use

  1. Open a webpage with images, videos, or audio.
  2. Tap the Firefox menu (⋮) and choose Download to Index from the extensions list.
  3. Pick the media you want to upload from the list.
  4. Add optional comma-separated tags and tap Upload.

Scroll the page first if items are still lazy-loading — the picker only finds media that's already in the page.

Privacy

  • Files and tags are sent only to the server URL you configure above.
  • The extension reads cookies for that server (to keep your session) and rewrites the Referer / Origin headers on requests it sends there (to satisfy CSRF checks).
  • No analytics, telemetry, or third-party requests.
  • Host access is requested per server when you save settings — the extension is not granted access to all sites at install.
